# YouTube Thumbnail Prompting Guide

This guide helps generate effective prompts for AI image models to create high-CTR YouTube thumbnails.

## ⛔ CRITICAL: NO TEXT IN IMAGES

**NEVER include text, words, letters, numbers, logos, watermarks, titles, labels, captions, or any written content in image prompts.** Text will be added separately by the design team. Every prompt MUST end with "no text, no words, no letters, no logos" to reinforce this.

## Core Principles of High-Performing Thumbnails

### 1. Visual Hierarchy
- ONE clear focal point that draws the eye immediately
- Maximum 2-3 visual elements total
- Empty space (negative space) creates breathing room
- Subject should occupy 40-60% of the frame

### 2. Contrast & Color
- High contrast between foreground and background
- Bright, saturated colors perform better than muted tones
- Red, yellow, and orange create urgency
- Blue and green create trust and calm
- Use complementary colors for maximum pop

### 3. Emotional Impact
- Human faces with clear emotions get 30% more clicks
- Exaggerated expressions work (surprise, shock, excitement)
- Eye contact with the viewer creates connection
- Body language should be dynamic, not static

### 4. Composition Rules
- Rule of thirds for subject placement
- Diagonal lines create energy and movement
- Leading lines guide the eye to the focal point
- Frame within a frame adds depth
